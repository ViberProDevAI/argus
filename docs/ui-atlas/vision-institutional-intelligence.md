# Institutional Intelligence Visual Manifest

## Design Thesis
- Ciddi, güven veren, veri-öncelikli görünüm
- Düşük gürültü, yüksek okunabilirlik
- Sinyal-riski-aksiyon hiyerarşisi

## Palette
- Background: #07090C
- Surface 1: #0F141B
- Surface 2: #151C26
- Primary: #3B82F6
- Positive: #16A34A
- Negative: #DC2626
- Warning: #D97706
- Text Primary: #E5E7EB
- Text Secondary: #9CA3AF
- Text Tertiary: #6B7280

## Card Scale
- NanoCard: tek metrik + trend
- MicroCard: 2 metrik + kısa context
- StandardCard: başlık + içerik + alt aksiyon
- InsightCard: gerekçe + güven + kaynak
- HeroCard: özet KPI + primary CTA
