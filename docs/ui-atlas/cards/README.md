# Card Atlas Outputs

Bu klasör, kart bazlı ekran görüntülerini saklamak için ayrıldı.

Planlanan dosya formatı:
- `card-<component-name>-before.png`
- `card-<component-name>-after.png`

Not:
- Kart listesi için `../card-matrix.md` dosyasını referans alın.
- Öncelik sırası: nano -> micro -> standard/insight kartlar.
